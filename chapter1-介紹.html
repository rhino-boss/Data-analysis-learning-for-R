1.1 Rstdio 介面簡介

打開 Rstudio，會顯示出類似於下面的界面，在常規的菜單欄與工具欄之下，主要有四個窗口，傳言說界面佈局與 MATLAB 非常像，不過我沒用過 MATLAB 所以我也不知道。

  • Source：用於編輯程式碼（如果未出現該窗口，用快捷 鍵Ctrl+Shift+N 新建一個 Untitled.R）。
  • Console：用於輸入命令。
  • Environment：為環境窗口，可以暫時簡單理解為查看變量的窗口。(右上)
  • History：為歷史窗口，用於查看執行過的歷史紀錄。(右上)
  • File：用於查看與管理文件、執行位置。(右下)
  • Plot：用於查看輸出的繪圖。(右下)
  • Packages：查看套件。(右下)
  • Help：查看套件的幫助文檔。(右下)

1.2 快捷鍵

描述 快捷鍵 使用位置
清空 Console Ctrl + L Console
產生箭頭 <- Alt + - Source
光標定位到 Source Ctrl + 1 任何位置
光標定位到 Console Ctrl + 2 任何位置
新增一個文件 Untitled.R Ctrl + Shift + N Source
在 Source 中查找或替換 Ctrl + F 任何位置
